## 📚 Student Management System (C++)

A console-based Student Management System implemented in C++ using STL containers like `vector` and algorithms such as `sort`.

### 🚀 Features
- ➕ Add new students with fixed subjects
- 🔍 Search students by name or ID
- ✏️ Update student grades with validation
- ❌ Delete students by ID
- 📊 Display student details (total, average, percentage)
- 🏆 Rank students based on total grades
- 🔢 Sort students by grades or ID

### 🛠️ Concepts Used
- Structs and vectors
- Functions and modular design
- STL algorithms (`sort`, `stable_sort`)
- Lambda expressions
- Input validation
- Pass by reference vs value

### 🎯 Notes
- Each student has a fixed set of subjects
- Grades are validated (0–100)
- Ranking supports tie-breaking logic
